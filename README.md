## ✨ Características

*   **Diseño Inmersivo:** Video de fondo que captura la esencia del río Magdalena.
*   **Exploración Interactiva:** Tarjetas con información detallada de lugares turísticos y gastronomía.
*   **Galería de Imágenes:** Ventanas modales con descripciones y fotos de cada lugar.
*   **Contenido Cultural:** Sección dedicada a la rica historia de El Banco.
*   **Mapa Integrado:** Mapa de Google Maps para una fácil ubicación.
*   **Totalmente Responsivo:** Experiencia de usuario óptima en cualquier dispositivo.

## 🚀 Cómo Empezar

1.  Clona este repositorio:
    ```bash
    git clone https://github.com/tu-usuario/banco-magdalena-turismo.git
    ```
2.  Navega al directorio del proyecto:
    ```bash
    cd banco-magdalena-turismo
    ```
3.  Abre `index.html` en tu navegador.
