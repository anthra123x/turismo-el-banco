Este proyecto es una guía turística interactiva para El Banco, Magdalena, diseñada para ayudar a los visitantes y locales a descubrir los tesoros culturales, gastronómicos e históricos de la región.

## ✨ Funcionalidades Principales

✔️ **Diseño Inmersivo:** Un video de fondo en el encabezado que captura la esencia del río Magdalena.  
✔️ **Secciones Detalladas:** Tarjetas interactivas para explorar lugares turísticos y la gastronomía local.  
✔️ **Galerías Modales:** Al hacer clic en una tarjeta, se abre una ventana con una descripción detallada y una galería de imágenes.  
✔️ **Guía Histórica:** Una sección dedicada a la rica historia y el legado cultural de El Banco.  
✔️ **Mapa Interactivo:** Un mapa de Google Maps integrado para ubicar fácilmente los puntos de interés.  
✔️ **Diseño Adaptable:** La página se ve increíble en cualquier dispositivo, ya sea móvil, tablet o de escritorio.

## 🏁 Cómo Empezar

Para ver el sitio en acción, simplemente clona este repositorio y abre el archivo `index.html` en tu navegador web preferido.

```bash
git clone https://github.com/tu-usuario/banco-magdalena-turismo.git
cd banco-magdalena-turismo
# Abre index.html en tu navegador
```
